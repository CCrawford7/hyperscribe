# How to Contribute

We welcome contributions to the Hyperscribe project!

## Getting Started

1.  Fork the repository.
2.  Clone your fork to your local machine.
3.  Create a new branch for your feature or bug fix.
4.  Load the extension in your browser using "Load unpacked" in `chrome://extensions`.

## Making Changes

-   Follow the existing code style and conventions.
-   Ensure any new features or changes are documented in the `/docs` wiki.
-   If you add new dependencies, update the relevant documentation.

## Submitting a Pull Request

1.  Push your changes to your fork.
2.  Open a pull request to the main repository.
3.  Provide a clear description of your changes and why they are needed.
4.  Link to any relevant issues.

Thank you for contributing!
